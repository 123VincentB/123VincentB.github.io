# vincentb.fr — Photometry & Metrology

**[vincentb.fr](https://vincentb.fr)** · mirror: [123vincentb.github.io](https://123vincentb.github.io/)

Personal blog by Vincent B., written from an ISO 17025 accredited photometry laboratory in Switzerland.

## Topics

- Photometric file formats (EULUMDAT)
- Light measurement: luminous flux, spectral power distribution.
- Measurement uncertainty and laboratory practice
- Open-source tools for photometry

## Stack

- [Hugo](https://gohugo.io/) static site generator with the [PaperMod](https://github.com/adityatelange/hugo-PaperMod) theme
- Deployed via GitHub Pages, served through Cloudflare

## Related projects

- [eulumdat-py](https://github.com/123VincentB/eulumdat-py) — Python library for reading and writing EULUMDAT (.ldt) photometric files, available on [PyPI](https://pypi.org/project/eulumdat-py/)
